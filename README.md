# Clientes API

API desenvolvida em ASP.NET Core 8 para gestão de clientes.

## Funcionalidades

- Autenticação JWT
- Listagem de clientes
- Consulta por Id
- Criação de clientes
- Atualização de saldo
- Transferência de saldo
- Validação de dados
- Lista em memória (Database)
- Docker

## Tecnologias Utilizadas

- .NET 8
- ASP.NET Core Web API
- JWT
- Swagger
- Docker

## Como Executar

```bash
dotnet run
```

Swagger:

```
https://localhost:5168/swagger
```
